# Numbers 1–100 — Attribute Table
_Last Updated: 2026-05-05_

## 1. Introduction
This page provides a structured classification of integers from 1 to 100.

---

## 2. Data Schema

### A. Column Definitions

* **n**: Integer value (1–100)
* **parity**: `even` or `odd`
* **prime**: 1 if prime
* **composite**: 1 if composite (n > 1 and not prime)
* **semiprime**: 1 if product of two primes
* **square**: 1 if perfect square
* **cube**: 1 if perfect cube
* **pow2**: 1 if power of 2
* **fib**: 1 if Fibonacci number
* **triangular**: 1 if triangular number
* **perfect**: 1 if perfect number
* **divisor_count**: total number of divisors
* **digit_sum**: sum of decimal digits
* **digital_root**: repeated digit sum (1–9)
* **feature_count**: number of active (1) feature flags
* **note**: short interpretive summary

---

### B. Design Policy

* **Zero-vector elimination:** Every number has at least baseline attributes
* **One-hot where appropriate:** Binary flags retained for key sets
* **Hybrid schema:** mix of categorical, numeric, and binary features
* **Pipeline-friendly:** deterministic and reproducible
